### Dictionaries and Loops
```python
# Formatting nested dictionaries - my breakthrough moment!
for key, value in data.items():
    if type(value) == dict:
        print(f"{key}:")
        for sub_key, sub_value in value.items():
            print(f"  {sub_key}: {sub_value}")
    else:
        print(f"{key}: {value}")
```

### Important Concepts Discovered
- Variables in loops are GENERIC (they receive each pair)
- Python cares about POSITION, not variable names
- Indentation is CRITICAL (especially with if/else)
- `type()` checking for handling different data structures
- Scope matters - can only use variables that exist!
